# The Challenge

Your task is to recreate the provided design (see the reference image) as closely as possible.

You will need to download the project, use the existing files, and complete the task based on them.

## Requirements

- Build the **desktop version** of the design using **HTML** and **CSS**.
- All design reference images are located in the **`design`** folder.
- Use all relevant **colors**, **fonts**, and **style details** found in the **`style-guide.md`** file.

## Getting Started

1. Clone or download the project files.
2. Open the project folder in your code editor.
3. Use the provided assets and follow the design to build the layout.
4. Open your HTML file in the browser to preview your work.

## Bonus
- Create the "Active states" design using HTML and css. You can find all of the active states design images inside design folder

- Use JavaScript to avoid clicking on submit before rating, add an error message too.

- Add the font that can be found inside assets/fonts to every text.
